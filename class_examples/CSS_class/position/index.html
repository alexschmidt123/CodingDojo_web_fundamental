CSS Position

FIRST RULE OF POSITION: DO NOT ABUSE POSITION!

The position property defineshow an element is positioned on your page.

There are five values for the position property.

  1. Static
  2. Relative
  3. Absolute
  4. Fixed
  5. Sticky

Static

The default value for position. The element is positioned normally according to your page flow.

Static Position

Relative

The element is positioned relative to its normal position.

Relative Position

Top: 50px; Left: 100px.

Absolute

The element is positioned relative to its nearest positioned ancestor

If you have the element absolutely positioned, you should relatively position their parent

Absolute Position

Fixed

The element is positioned relative to the viewport.

Position Fixed

Sticky

The element toggles between relative and fixed, depending on scoll position

Sticky
